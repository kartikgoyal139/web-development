Html 5 Game Development

The course of HTML 5 Gam edevelpment has been offered to the Second Year Student of the VIT Bhopal University.

Course Objectives.

This course objectsives are as follows:

  1. To learn the basics of web game development from scratch
  2. To expose the salient techniques to render the games for mobile platforms and web browsers
  3. Focus on modern game platforms, differences, evolution, and limitations, on game programming
  4. Identify best practices for game development in browsers.

Course Outcomes

The students who complete the course will be able to:

  • Create a static web page in HTML and CSS.
  • Add functionalities to page elements
  • Create a basic game using JavaScript libraries.
  • Integrate physics to easily in Web based Games
  • Create platform games, puzzle and card games.
  • Deploy games on multiple platforms.
